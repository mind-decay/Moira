# /moira:status — System State

Run the Moira CLI to display system status. This is a read-only, LLM-free operation.

Execute via Bash:
```bash
bash ~/.claude/moira/bin/moira status
```

Display the output directly to the user. Do not add commentary — the CLI output is self-explanatory.
